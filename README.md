🔥 لمحة عامة

OSINT RAVEN هو منصة متكاملة لجمع المعلومات من المصادر المفتوحة (Open Source Intelligence) مصممة خصيصًا للجهات الأمنية المرخصة وخبراء الاختراق الأخلاقي. يدمج النظام بين قوة واجهات برمجة التطبيقات الرسمية (API) وذكاء المتصفحات الخفي (Stealth Crawling) وتقنيات المحاكاة البشرية لاستخراج البيانات المخفية والمحمية، مع إمكانية تضمين وحدة محاكاة للتصيد (Phishing) لأغراض التدريب والتعليم.

⚠️ تنبيه قانوني حاسم: هذا النظام مخصص للاستخدام التعليمي والاختبارات الأمنية المرخصة فقط. استخدامه للوصول إلى حسابات أو بيانات دون إذن صريح يعتبر جريمة معلوماتية يعاقب عليها القانون. أنا، حسان المريسي، أتبرأ من أي استخدام غير قانوني.

🚀 المميزات والقدرات الهجومية

· جمع شامل من منصات ميتا (Facebook, Instagram) عبر واجهات API الرسمية مع إدارة الحصص والبروكسيات.
· متصفح خفي (Stealth Crawler) يعتمد على Playwright لمحاكاة السلوك البشري، وتجاوز أنظمة كشف البوتات، واستخراج:
  · الأرقام والإيميلات المخبأة في البايو والمنشورات.
  · روابط الحسابات المرتبطة.
  · المحتوى المؤقت (Stories) بعد تسجيل الدخول (باستخدام بيانات مخترقة).
· وحدة تصيد احترافية (Phishing Server):
  · خادم Flask يقدم صفحة دخول مطابقة لفيسبوك.
  · تسجيل آمن للبيانات المسروقة (IP، وكيل المستخدم، البريد، كلمة المرور).
  · دعم SSL (HTTPS) لتبدو شرعية.
· وحدة سرقة بيانات المتصفحات (Browser Credential Stealer):
  · استخراج كلمات المرور المحفوظة من كروم وفايرفوكس (لأغراض تعليمية).
  · فك التشفير باستخدام مفتاح النظام (محاكاة).
· ربط الكيانات (Entity Resolution):
  · توحيد الحسابات عبر المنصات بناءً على الإيميل، رقم الهاتف، والاسم.
  · كشف العلاقات الخفية بين الأفراد.
· تحليل الشبكات (Graph Analysis):
  · بناء رسم بياني تفاعلي باستخدام NetworkX.
  · حساب مركزية العقدة (Degree, Betweenness) لتحديد المؤثرين.
  · اكتشاف المجموعات (Communities) باستخدام خوارزمية لوفان.
  · تحديد العناقيد المشبوهة (كثرة الحسابات الخاصة ذات المركزية العالية).
· خرائط مرئية تفاعلية:
  · رسوم بيانية ثابتة (PNG/SVG) باستخدام Matplotlib.
  · خرائط ويب تفاعلية (HTML) باستخدام Plotly، قابلة للتصدير والعرض.
· تقارير ذكية:
  · تقرير HTML كامل يضم جميع الكيانات والعلاقات والصور البيانية.
  · جاهز للعرض على الجهات المختصة.

🛠️ المتطلبات التقنية

· نظام تشغيل: Linux / macOS / Windows (مع WSL2)
· لغة البرمجة: Python 3.9+
· المكتبات: موجودة في ملف requirements.txt
· متصفحات: Chromium (سيتم تثبيته عبر Playwright)

---

📦 التثبيت والإعداد

1. استنساخ المستودع:
   ```bash
   git clone https://github.com/hassan-almarisi/osint-raven.git
   cd osint-raven
   ```
2. تثبيت الحزم:
   ```bash
   pip install -r requirements.txt
   ```
3. تثبيت متصفحات Playwright:
   ```bash
   playwright install chromium
   ```
4. إعداد الشهادات للتصيد (اختياري):
   ```bash
   mkdir ssl
   openssl req -x509 -newkey rsa:4096 -keyout ssl/key.pem -out ssl/cert.pem -days 365 -nodes
   ```
5. ضبط المتغيرات البيئية:
   · أنشئ ملف .env في المجلد الرئيسي وأضف:
     ```
     FB_TOKEN=your_facebook_access_token
     IG_TOKEN=your_instagram_access_token
     ```
   · أو قم بتعيينها كمتغيرات بيئة في نظامك.

⚙️ ملف الإعدادات (config.py)

يمكنك تعديل الإعدادات التالية:

· PROXY_LIST: قائمة بروكسيات (HTTP/SOCKS5) لتدوير العناوين.
· USER_AGENTS: قائمة وكلاء مستخدم عشوائيين.
· إعدادات خادم التصيد (المنفذ، استخدام SSL، المسارات).

🧠 طريقة الاستخدام

لتشغيل النظام بالكامل:

```bash
python main.py
```

سيقوم النظام بـ:

1. جمع البيانات من فيسبوك وانستغرام عبر API.
2. تشغيل المتصفح الخفي لزيارة روابط محددة (عدّل الكود حسب هدفك).
3. تشغيل خادم التصيد (إذا فعّلته) في خلفية منفصلة.
4. محاولة استخراج بيانات المتصفحات المحلية.
5. ربط الكيانات وتحليل الشبكة.
6. إنشاء الرسوم البيانية والتقارير في مجلد data/.

يمكنك تشغيل كل وحدة على حدة باستيرادها من المجلد modules.

```
OSINT_RAVEN/

├── config.py                # الإعدادات والمتغيرات

├── modules/                 # جميع الوحدات

│   ├── phish_server.py      # خادم التصيد
│   ├── cred_stealer.py      # سارق بيانات المتصفحات
│   ├── entity_resolver.py   # محلل الكيانات
│   ├── graph_analyzer.py    # محلل الشبكات
│   ├── visualizer.py        # مصور البيانات
│   ├── reporter.py          # مولد التقارير
│   └── utils.py             # دوال مساعدة
├── data/                    # مخرجات النظام (صور، تقارير)
├── logs/                     # سجلات النظام والتصيد
└── ssl/                      # شهادات HTTPS للتصيد
```
